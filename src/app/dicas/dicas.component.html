Como importar meus snnipets de código no Visual Studio Code
  1. localize o diretório no seu computador
    • C:\Users\SEU_USUARIO_AQUI\AppData\Roaming\Code\User\snippets
  2. Baixe o meu snnipet e cole nesse diretório
  3. Exemplo de uso:
    1. Digite oliota. em uma página html no visual studio code
    2. Escolha oliota.card.simples
  4. Use TABS para preencher as informações
  5. Organize o código para identar os novos elementos criados
Como criar um arquivo de snnipets de códigos para o Visual Studio Code
  1. No visual studio code execute o atalho
    • Ctrl + Shift + p
      1. Busque por : Preferences - configure user snnipets
      2. New Global snnipets
      3. Escolha um nome para o arquivo (não pode conter espaços e nem caracteres especiais)
  2. O arquivo gerado está parcialmente comentado 
    • {{exemplo}}
  3. Exemplo de uso:
    1. Digite oliota. em uma página html no visual studio code
    2. Escolha oliota.div
Como criar um resumo de perfil no Github
  1. Acesse sua conta no github
  2. Crie um repositorio com o mesmo nome do seu usuario e marque a opção (Create README file)
  3. Ao fazer isso a página inicial do seu perfil no github já tera um resumo
  4. Para editar o resumo será necessário editar o README desse repositorio
  5. É possivel fazer de 2 formas
    1. Editando manualmente no github
    2. Clonando o projeto e fazendo a alteração no Visual Studio Code (recomendado)
      • Fazer o clone do repositorio
      • Editar o README (use algum preview) para ajudar
      • Realize o commit e faça o push
      • Ao fazer isso as mudanças já serão visiveis no seu perfil do Github
        Segue um template
        • Meu perfil no github
        • Repositorio com mesmo nome do meu perfil
        • Arquivo de exemplo para copiar o código
        • OBS: Onde tiver username=oliota , coloque o seu usuario
Como clonar um repositorio do Github no computador
  1. Baixar e instalar o Git
  2. Acesse sua conta no github
  3. Acesse um repositorio
  4. Selecione o botão verde CODE
  5. Copie o endereço que aparece na janela que apareceu após clicar no botão CODE
  6. Escolha uma pasta no computador para clonar o repositorio
  7. Execute um click com o botão direito do m ouse e escolha "git bash here"
  8. Na janela do terminal digite git clone e cole o endereço do repositorio aqui
  9. Tecle ENTER para confirmar
  10. Após o comomando executar será criada uma pasta com o mesmo nome do repositório no computador
Como salvar as mudanças feitas no computador para o repositorio no github
  1. Localize a pasta principal do seu repositorio clonado no computador
  2. Clique com o botão direito do mouse na pasta principal (ou se a pasta já estiver aberta clique com o botão direito do mouse em um espaço em branco da pasta)
  3. Escolha git GUI here
  4. Após abrir a janela selecione o botão Staged changed
  5. Ao clicar os arquivos serão migrados da lista vermelha para lista verde, isso significa que os arquivos serão considerados no commit
  6. Agora no campo commit, informe uma mensagem que justifique sua alteração, pode ser uma frase curta
  7. Ao informar uma mensagem de commit é possivel clicar no botão COMMIT
  8. Após clicar no botão commit a tela ficará em branco, isso significa que o pacote já esta pronto para fazer um push
  9. Selecione o botão push
  10. Na janela que foi aberta selecione novamente o botão push
  11. Na primeira vez que é realizado um push na maquina será solicitado as credenciais do git
  12. Escolha a 1º opção "Autenticar no navegador" e utilize o mesmo navegador onde o github esta conectado para facilitar
  13. Conceda acesso ao seu github e pronto, as suas mudanças estarão salvas no github
  14. Para conferir, acesse o seu repositorio no site do github e localize as suas mudanças
  15. Acesse sua conta no github
