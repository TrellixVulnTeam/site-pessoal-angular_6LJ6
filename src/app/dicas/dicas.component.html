Como importar meus snnipets de código no Visual Studio Code
  1. localize o diretório no seu computador
    • C:\Users\SEU_USUARIO_AQUI\AppData\Roaming\Code\User\snippets
  2. Baixe o meu snnipet e cole nesse diretório
  3. Exemplo de uso:
    1. Digite oliota. em uma página html no visual studio code
    2. Escolha oliota.card.simples
  4. Use TABS para preencher as informações
  5. Organize o código para identar os novos elementos criados
Como criar um arquivo de snnipets de códigos para o Visual Studio Code
  1. No visual studio code execute o atalho
    • Ctrl + Shift + p
      1. Busque por : Preferences - configure user snnipets
      2. New Global snnipets
      3. Escolha um nome para o arquivo (não pode conter espaços e nem caracteres especiais)
  2. O arquivo gerado está parcialmente comentado 
    • {{exemplo}}
  3. Exemplo de uso:
    1. Digite oliota. em uma página html no visual studio code
    2. Escolha oliota.div
