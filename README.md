# OliotaAngular

Novo site pessoal desenvolvido em angular

## Subindo o servidor

execute o comando `ng serve --port 5000` . Link para testar `http://localhost:5000/`.

## Build

Execute `ng build` o conteudo sera gerado no diretorio `dist/`
